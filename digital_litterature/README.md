# Litterature numérique

## Projet

### Love Letter

Mon programme est une reproduction du code de Nick Montfort en Java.
Les lettres s'écrivent dans la console et j'ai essayé de les traduires en français. Malheuresement, le code a ses limites et a été prévu pour l'anglais donc l'accord avec les noms ne sont pas correct.

### Bad Apple

Mon programme permet reproduire le clip **_Bad Apple_**.

Quand on appuie sur "espace", le programme se met en pause.

> Remarque : Je me suis basé sur ce code : https://github.com/IntangibleMatter/badapple_processing. Je l'ai relu complètement et adapté pour retirer les cases noirs et blanche et mettre l'intégralité des paroles en tant que couleur. La fonction de pause a aussi été ajouté pour des raisons pratiques.

> Bibliothèque : Ce programme utilise la librairie sound : https://processing.org/reference/libraries/sound/index.html.

## Installation

Une fois **_Processing_** installé, ouvrez le fichier et executez le.
