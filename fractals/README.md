# Fractals

## Projet

Mon programme permet de dessiner la fractale du triangle de Sierpiński.

Quand on appuie sur "+" ou "-", on augmente le nombre d'itérations.

Quand on click, ça change de couleur aléatoirement.

> Remarque : Après avoir énormément galérer à faire la fractale, j'ai décidé de supprimer ce que j'avais fait et qui ne marchait pas. Je suis repartis sur de bonnes bases grâce à l'assistance de l'IA pour ensuite modifier certaines parties et ajouter de l'intéractivité.

## Installation

Une fois **_Processing_** installé, ouvrez le fichier et executez le.
