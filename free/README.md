# Libre

## Projet

Mon programme permet de générer un arbre en fractale qui grandit en fonction de la pluie qui lui tombe dessus. L'angle de rotation évolue en fonction de l'amplitude de la musique joué.

J'ai mis dans le dépôt deux musiques, vous pouvez la changer en modifiant la ligne 34.

```
  music = new SoundFile(this, "nothing_else_matter.mp3");
```

## Installation

Une fois **_Processing_** installé, ouvrez le fichier et executez le.

> Bibliothèque : Ce programme utilise la librairie sound : https://processing.org/reference/libraries/sound/index.html.
